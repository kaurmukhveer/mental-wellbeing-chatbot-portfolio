# mental-wellbeing-chatbot-portfolio
Architecture and engineering showcase for an AI-assisted mental wellbeing web application built using React, Node.js, Supabase, and Retrieval-Augmented Generation (RAG).

## Repository Notice

This repository is a public engineering portfolio showcase for an academic honours project.

Source code, deployment configurations, datasets, and confidential implementation details are intentionally excluded due to NDA and academic project restrictions.

The repository focuses on:
- system architecture
- engineering decisions
- workflow design
- feature demonstrations
- technical documentation
